Keyboard Shortcuts

  • SHIFT: Turns off grid for freeform placement/movement
  • SPACE: Temporary MOVE mode () while held
  • F: Toggles node under the mouse's final status
  • I: Toggles node under the mouse's initial status
  • D: Enter DRAW mode ()
  • M: Enter MOVE mode ()
  • E: Enter ERASE mode ()
  • CTRL: Temporarily freezes moved control points
  • SHIFT + SPACE: Insert blank character in test string
